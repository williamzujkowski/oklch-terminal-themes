# site

GitHub Pages theme picker for [`@williamzujkowski/oklch-terminal-themes`](..).

Built with [Astro 5](https://astro.build/). Deployed to https://williamzujkowski.github.io/oklch-terminal-themes/.

## Dev

```bash
# From repo root
pnpm install
pnpm --filter @williamzujkowski/oklch-terminal-themes-site dev
```

## Build

```bash
pnpm --filter @williamzujkowski/oklch-terminal-themes-site build
```

Output in `site/dist/`. CI deploys this via `.github/workflows/pages.yml`.
